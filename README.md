# Empik App

>This is recruitment app for Empik.

### Used Tech Stack:
>* Java 17
>* Spring 6, and Spring Boot 3
>* PostgreSQL
>* Testcontainers, mockito and JUnit5


### How to run Application
> ```shell
> docker compose up --build
> ```

### How to run tests
> ```shell
> docker-compose -f docker-compose-test.yml up --build
> ```
